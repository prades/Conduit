PYLONS
Cost 10 shards · 30s construction time
CORE followers Build pylons instantly (no wait)
Place Toggle BUILD ON → tap any valid tile
Element upgrade Long-press pylon → radial LEFT to change element
PYLON MODES
Attack Mode: Single-element tower. Fires at enemies within detection range. Assign an element to activate.
Wave Mode: Place 2 adjacent pylons of the same element → they pair up and project a zone effect corridor between them.
NETWORK STRENGTH
Connect same-element pylons into a group to gain network buffs. Larger groups = stronger tiers.
T1 2+ pylons connected — basic network bonus
T2 4+ pylons connected — enhanced zone effects
T3 6+ pylons connected — maximum network power
NETWORK INTEGRITY
A 0–100 meter that builds while your network is active and intact.
Breaking connections causes it to decay. Higher integrity amplifies zone effects.
SEASONED BONUS
+25% power Awarded to any pylon that survives a full night wave without being destroyed
